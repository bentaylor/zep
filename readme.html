Hello There!

Thank you for taking the time to try out Pez. We (Lee and Ben) think you're super awesome for doing so. Let's get you set up.

Now that you have Pez downloaded, all you need is your own domain (http://www.thisisyou.com) with php5 enabled.

How To Install

  1. Think of a strong password. Got it? Good!
  2. Open the "pz-config" file. See where the word, "password" is lower-cased? Replace it with your password.

    (Note: Yes, this file will be on your web server. Pez checks this file to authenticate your password so you can log into the admin section. This file can only be viewed by you when logged into your web server. No one else can see it.)
  3. Decide what directory you want Pez to run from, and drop it in.
  4. Once uploaded, set the "cache" directory to chmod 777.
  5. Navigate to http://www.thisisyou.com/admin/ and log in using your password.
  6. Once in, start entering the RSS feeds from the services you use (Flickr, last.fm, digg etc.)

If you have any questions you can e-mail us pez@bogdind.com or twitter us @pezapp.

Pez is an open-source project by Lee Kelleher and Ben Taylor. It is licensed under the LGPL. The concept was originally based on the Personal Zeitgeist software by Creative Synthesis. It uses SimplePie to aggregate the RSS feeds.
